---
* 一般业界产品完整的文档包括：  
   * 1. 产品需求，详细拆分（对内/外）  
   * 2. 详细设计（研发技术文档，对内）
   * 3. 产品使用说明（对外）
      * 3.1 产品使用操作说明书（硬件配置）
      * 3.2 培训教材，示例tutorial；
      * 3.3 软件对外接口调用说明
   * 4. 使用过程Q/A问题集锦（包括软件返回值错误码表）（对外）
   * 5. 测试文档（对内详细流程/对外展示结果）
   * 6. 第三方机构认证/测试 证书（根据市场和客户需求）
